# AGENTS

本仓库的项目管理入口文件，用于帮助新的 GPT/协作者快速了解需要维护的管理文件、约定与当前方向。

## 目录/文件清单
- **task.md**：当前任务清单（可执行的具体事项）。
- **todos.md**：待办列表（零散或未成形的想法/需求）。
- **plans.md**：阶段性计划与里程碑（路线图）。
- **progress.md**：进度与日志（按日期记录阶段性成果）。
- **findings.md**：调研/探索记录（参考资料、对比分析、灵感收集）。
- **decisions.md**：关键设计/技术决策与原因（ADR 风格简化版）。
- **notes.md**：临时笔记与会议要点。

> 如需新增管理文件，请在此处补充，并说明用途与与其他文件的边界。

## 维护约定
- 首次进入仓库请先阅读 `agents.md`。

- 每次完成阶段性工作，请在 `progress.md` 更新一条记录，并在必要时同步 `decisions.md`。
